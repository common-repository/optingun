== Description ==

Optingun helps to create lead capture forms in minutes and this plugin will help integrating those forms to your wordpress site, it also integrates more than 30 auto-responders to send emails to your leads and help automate the process.

Major features in Optingun include:

* Lead capture forms
* social signup forms with just one clicking
* integrations to all email softwares(30 plus)
* more than 80 form templates to choose and customisable designs
* no limit on features, all features from free accounts


PS: Only the users of Optingun.com can use this plugin, you will need to enter your optingun api key.

== Installation ==

Upload the Optingun plugin to your blog, Activate it, then enter your Optingun API key [https://optingun.com/dashboard/api.php]

1, 2, 3: You're done!
